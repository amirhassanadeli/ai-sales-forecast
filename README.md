# AI Sales Forecast - Rossmann Dataset

This project is an end-to-end Machine Learning pipeline for predicting retail store sales using historical data.

## Project Goal

The goal of this project is to forecast daily sales for retail stores using machine learning techniques.

Accurate sales forecasting helps businesses:

- Improve inventory planning
- Optimize promotions
- Reduce operational costs

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Joblib

---

## How to Run

1. Install dependencies:

pip install -r requirements.txt

2. Run the full pipeline:

python main.py

---

## Machine Learning Pipeline

1. Data preprocessing
2. Feature engineering
3. Time-based train/test split
4. Model training
5. Model evaluation
